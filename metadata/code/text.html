Text Format¶

Code Metadata¶

Code Metadata items appear in the text format as custom annotations, and are considered attached to the first instruction that follows them.

\[\begin{split}\begin{array}{llclll} \def\mathdef136#1{{}}\mathdef136{code metadata annotation} & \href{text.html#text-codemetadata}{\mathtt{codemetadataannot}}(\mathtt{T}) &::=& \def\mathdef180#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef180{(@metadata.code.T}~\mathit{data}{:}\mathtt{T}~\def\mathdef181#1{\mbox{‘}\mathtt{#1}\mbox{’}}\mathdef181{)} \\ \end{array}\end{split}\]

Where T is the format name of the item, and data is a byte string containing the same payload as in the binary format.
